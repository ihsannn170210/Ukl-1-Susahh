# Program untuk Mengihitung Rata Rata Nilai
Program ini digunakan untuk menghitung nilai rata rata dari beberapa siwa ujian 
1. Program meminta pengguna memasukkan jumlah siswa.
2. Untuk setiap siswa:
- Meminta nama siswa.
- Meminta nilai UTS dan UAS.
- Menghitung rata-rata dari kedua nilai tersebut.
2 Setelah semua data dimasukkan, program menampilkan tabel berisi:
- Nama siswa
- Nilai UTS
- Nilai UAS
- Rata-rata nilai
